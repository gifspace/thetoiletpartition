# The Toilet Partition Website (MHC Group Products)

Project for a premium, high-performance static HTML website for **MHC Group Products Co., Ltd.**

## Folder Structure
- `/` - Root directory containing the 5 main HTML pages.
- `/assets/css/` - Custom design tokens and premium styling (`style.css`).
- `/assets/js/` - Interactive logic and scroll animations (`main.js`).
- `/assets/images/` - Optimized project and product imagery.

## Technology Stack
- **Framework**: [Tailwind CSS](https://tailwindcss.com) (via CDN) for modern, responsive layouts.
- **Interactivity**: [Alpine.js](https://alpinejs.dev) for lightweight components (FAQs, Mobile Menu).
- **Typography**: Google Fonts (Inter & Sarabun for Thai support).
- **Animations**: Custom Intersection Observer API for scroll-triggered reveals.

## Page Descriptions
1. **Home (`index.html`)**: Features the Hero Hook, Product Series, Problem/Solution grid, and FAQ.
2. **About Us (`about.html`)**: Details MHC Group profile, Vision/Mission, and Core Values.
3. **Products (`products.html`)**: Full breakdown of HPL, PB, and MFF series with a technical comparison table.
4. **Works (`works.html`)**: Filterable project gallery showcasing 8+ major installations.
5. **Contact Us (`contact.html`)**: Premium multi-channel contact page with an interactive form.

## Deployment Instructions
This is a **Static HTML** project. To deploy:
1. Upload all files from the root directory to your web server (public_html).
2. Ensure the `/assets` folder is uploaded along with the HTML files.
3. The site is compatible with **Apache 2.4** and **PHP 7.4** environment as requested.
