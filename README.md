# The Toilet Partition (MHC Group Products)

Official website for The Toilet Partition by MHC Group Products.

## Features
- Premium Product Showcase (HPL, PB, MFF Series)
- Detailed Technical Specifications
- Project Portfolio
- Responsive Modern Design

## Tech Stack
- HTML / Vanilla CSS
- Alpine.js
- Tailwind CSS (via CDN)
- Font: Anuphan / Inter

## Development
- `generate-products.js`: Script to generate product detail pages from source text data.
- `index.html`: Main landing page.
- `products.html`: Product archive with filtering.

## Hosting
Deployed via GitHub Pages.
